简单的例子

如果你已经下载好整个框架的源码，那么你可以在这里找到应用的例子:

plainframework/applications/pf_simple

如果你在windows上工作，可以使用vs2008以上版本打开pf_simple/scripts/pf_simple.vc9.sln进行编译，如果你在linux平台下则使用cmake进行编译：

cd plainframework/applications/pf_simple #进入示例的目录
cmake ./
make

下面是例子的实现代码，非常简单，如果开启了网络模块则会报错，后面的网络模块中会详细的介绍到。

实现代码

#include "pf/engine/kernel.h"
#include "main.h"

pf_engine::Kernel g_engine_kernel;
int32_t main(int32_t argc, char * argv[]) {
#if __WINDOWS__
  USE_PARAM(argc);
  USE_PARAM(argv);
  _CrtSetDbgFlag(_CrtSetDbgFlag(0) | _CRTDBG_LEAK_CHECK_DF);
  system("color 02"); //color green
  system("mode con cols=120"); //cmd size
  WORD versionrequested;
  WSADATA data;
  int32_t error;
  versionrequested = MAKEWORD(2, 2);
  error = WSAStartup(versionrequested, &data);
#endif
  //g_engine_kernel.setconfig(ENGINE_CONFIG_NET_ISACTIVE, true);
  g_engine_kernel.init();
  g_engine_kernel.run();
  g_engine_kernel.stop();
  return 0;
}
